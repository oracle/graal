LLVM Upstream Patches
---------------------

This directory contains patches which were used to build this
LLVM distribution but are not yet upstream or have been backported
to LLVM 22. To build this LLVM distribution yourself, apply the patches
on top of an LLVM [22.1.8](https://github.com/llvm/llvm-project/tree/llvmorg-22.1.8) source tree.
